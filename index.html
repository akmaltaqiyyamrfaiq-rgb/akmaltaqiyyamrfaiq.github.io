Tentang Saya

Saya Akmal, siswa SMK jurusan Teknik Komputer dan Jaringan. Saya memiliki ketertarikan dalam bidang administrasi sistem, jaringan komputer, dan pengembangan web dasar. Saat ini saya fokus mempelajari instalasi sistem operasi, konfigurasi jaringan lokal (LAN), dan simulasi topologi jaringan.

Saya percaya bahwa kemampuan teknis harus dibarengi dengan pemahaman konsep yang kuat dan pengalaman langsung. Oleh karena itu, portofolio ini menjadi tempat saya mendokumentasikan proses pembelajaran dan proyek latihan yang saya kerjakan selama masa studi.

Keahlian

  • Pemahaman dasar jaringan komputer (LAN, IP Address, dan topologi jaringan)
  • Instalasi dan konfigurasi sistem operasi (Linux Debian, Windows)
  • Konfigurasi IP Address pada perangkat jaringan
  • Pemahaman dasar penggunaan Cisco Packet Tracer
  • Dasar pemrograman web (HTML dan CSS)
  • Kemampuan troubleshooting sederhana pada perangkat komputer

Proyek

Latihan Instalasi Sistem Operasi Linux Debian

Melakukan proses instalasi dan konfigurasi dasar sistem operasi Linux Debian sebagai latihan pembelajaran administrasi jaringan di sekolah.

Konfigurasi IP Address

Mempelajari dan mempraktikkan cara mengatur IP Address pada jaringan lokal untuk memahami konsep dasar komunikasi antar perangkat.

Pengenalan Jaringan LAN dan Simulasi Topologi

Melakukan simulasi sederhana jaringan LAN menggunakan software pembelajaran untuk memahami struktur dan fungsi dasar switch dan router.
